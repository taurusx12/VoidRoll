# Phase 11 Command Notes — Secret / Voidborn Reveal

## Commands affected

### /roll
When rarity is:
- SECRET: use SECRET_BOTTOM_UP reveal.
- VOIDBORN: use VOIDBORN_BOTTOM_UP reveal.
- DIVINE: use DIVINE_REVEAL.

### /pack
Same reveal logic as /roll.

### /banner pull
Same reveal logic as /roll.

## Reveal behavior

For SECRET:
1. Send quote embed first.
2. Wait around 1.2 seconds.
3. Send final character card embed/image.

For VOIDBORN:
1. Send Voidborn reveal embed.
2. Send final character card.
