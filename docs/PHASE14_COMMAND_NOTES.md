# Phase 14 Command Notes — Dungeons

## /dungeon
Starts a dungeon.

Options:
- type: normal / elite / abyss / void

## /dungeon-status
Shows current dungeon run.

## /dungeon-choose
Chooses path when a choice room appears.

Options:
- path: safe / balanced / danger / void

## /dungeon-abandon
Abandons current dungeon run.

## Important
Dungeons use:
- Battle Engine
- Formation System
- Rewards System

Do not add:
- Item Rolls
- Relic Pulls
- Aura Pulls
- Stars
- Fusion
