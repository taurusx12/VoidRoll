# Phase 6 Command Notes — Anime Database

## /anime
Shows anime database summary.

Examples:
- /anime Bleach
- /anime Chainsaw Man
- /anime Naruto

Shows:
- Total characters
- Rarity distribution
- Role distribution
- Element distribution

## /collection
Shows player anime completion.

Example:
- /collection anime:Bleach

Shows:
- Completion %
- Owned unique characters
- Total database characters
- Next milestone

## /character
Shows public character database entry.

Example:
- /character Makima

Shows:
- Name
- Variant
- Anime
- Rarity
- Element
- Role
- Base Power
- Passive
- Lore if available

## /who-has
Foundation command.

Example:
- /who-has Makima

Shows:
- Players who own the character
- Variant
- Rarity
- Level
- Power

## Important
- Completion counts unique character database entries, not duplicate copies.
- Variants count separately if they are separate characters.
- No stars.
- No fusion.
- No character-specific shards.
