# VoidRoll Big Update Full Package

هذا مو باتش سحبات فقط. هذا ملف شامل لكل أفكار الـBig Update المتفق عليها:

## الأنظمة المطلوبة

1. Multiple Active Banners
- أكثر من بانر بنفس الوقت.
- كل بانر له وقت خاص.
- لا يوجد Anime Pack.
- لا يوجد Secret Pack دائم.
- /banner يعرض كل البانرات الحالية.
- /pack banner:<id> يسحب من بانر محدد.
- كل سحبة = 10 Tokens.
- 20 سحبة = 200 Tokens = Guaranteed Featured.

2. Manual Fusion
- الدمج ليس تلقائي.
- /fuse name:<character>
- /fuse-list
- النجوم من ⭐1 إلى ⭐10.
- يزيد الباور مع كل نجمة.

3. No ID Gameplay
كل شيء باسم الشخصية:
- /t name:<character>
- /a name:<character>
- /fuse name:<character>
- /team set name:<character>
- /sell name:<character>

4. Infinite Inventory
- لا يوجد cap.
- إذا عند اللاعب 500 كرت، يظهر Card 1/500.
- التخزين مفتوح.

5. Multi Team Campaign
- Chapter 1-9 = فريق واحد.
- Chapter 10-19 = فريقين.
- Chapter 20-34 = 3 فرق.
- Chapter 35-49 = 4 فرق.
- Chapter 50-59 = 5 فرق.
- Chapter 60+ = 6 فرق.
- العدو يملك نفس عدد الفرق.

6. Class/Element Towers
- Dark Tower
- Light Tower
- Fire Tower
- Ice Tower
- Shadow Tower
- Curse Tower
- Hero Tower
- Demon Tower
كل تاور يسمح بعناصره فقط.
كلما زاد الفلور، يزيد عدد الفرق المطلوبة.

7. Real Dungeon
الدنجن ليس تاور:
- Wave Room
- Trap Room
- Treasure Room
- Heal Room
- Elite Room
- Final Boss

8. Boss Rush
- Solo Boss Rush.
- Global Boss Rush.
- الجوائز حسب Damage milestones.
- مدة البوس محدودة.
- Rewards حسب الدمج.

9. Resources
مو بس Gold:
- Essence Stones
- Divine Crystals
- Shadow Cores
- Ascension Flames
- Dungeon Keys
- Raid Tickets
- Trait Rerolls
- Evolution Gems
- Weapon Shards

10. Level System
- كل الشخصيات تقدر توصل Lv99.
- مافي cap للفل حسب الريرتي.
- لكن scaling يختلف حسب الريرتي.
- Common Lv99 لا يساوي Secret Lv99.

11. Stats / Attributes
كل شخصية عندها:
- ATK
- DEF
- HP
- SPD
- Crit Rate
- Crit Damage
- Element
- Class
- Passive
- Ultimate

12. Team Synergies
كل الشخصيات اللي بينها علاقة فعلية في الأنمي تعطي Buff:
- Team
- Family
- Rival
- Master/Student
- Organization
- Crew
- Duo
- Trio
- Villain Group

13. Equipment Effects
كل درع/سلاح يعطي خاصية:
- Dodge
- Shield
- Counter
- Ult Charge
- Crit
- Element Damage
- Defense Break

14. Auto Progress
- /auto-story
- /auto-tower
- /auto-dungeon
يلعب تلقائي لين يخسر.

15. Token Reward Rule
- التوكن كل 5 مراحل فقط.
- ليس كل مرحلة.
- العدد يزيد حسب تقدم اللاعب.

16. Admin Commands
- /admin-reset-all confirm:YES
- /admin-give-gold
- /admin-give-tokens
- /admin-give-resource
- /admin-give-rolls

17. Auction House
- /ah sell
- /ah browse
- /ah buy
- /ah cancel
- فلترة بالريرتي والنجوم والسعر والباور.
