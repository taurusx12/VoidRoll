# Phase 9 Command Notes — Character Evolution Tree

## /character-tree
Shows the selected card's full evolution tree.

Recommended options:
- card: owned card

Shows:
- Core Tree
- Skill Tree
- Gear Tree
- Trait Tree
- Bond Tree
- Transformation Tree
- Variant Tree

## /upgrade
Upgrades one branch.

Recommended options:
- card: owned card
- branch:
  - core
  - skill
  - gear
  - trait
  - bond
  - transformation
  - variant

## Important
Do not add:
- /fuse
- /star-upgrade
- /item-roll
- /character-shards

## Upgrade resources
Use:
- Gold
- Essence
- Soul Fragments
- Role Sigils
- Element Cores
- Rarity Materials
- Gear Materials
- Trait Materials
- Bond Points
- Transformation Cores
- Void Crystals
