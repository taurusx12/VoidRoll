# VoidRoll Reborn — Phase 12 Banner Rework

## الهدف
إعادة بناء نظام الرول والبانرات على النظام الجديد.

## القواعد
الرولات للشخصيات فقط.

محذوف:
- Item Rolls
- Relic Pulls
- Aura Pulls
- Fusion
- Stars

## أنواع الرول

### Normal Roll
للشخصيات العامة.

### Featured Banner
بانر بشخصية مميزة.

مثال:
Corrupted Makima — SECRET

### Limited Banner
بانر محدود يحتاج Void Crystals + Tokens.

## VOIDBORN
تم تثبيت VOIDBORN بين DIVINE و SECRET.

الترتيب:
COMMON → RARE → EPIC → LEGENDARY → MYTHIC → DIVINE → VOIDBORN → SECRET

- VOIDBORN Hard Pity
- SECRET Soft Pity
- SECRET Hard Pity

## Reveal Integration
إذا السحبة طلعت:

## Featured Character

ومع نظام الضمان، إذا ما حصلت Featured في مرة، تزيد فرصتها أو تضمن لاحقًا.
