# VoidRoll Reborn — Phase 3 Updated Resources

## التعديل المهم
تم حذف فكرة Character Shards بالكامل.

لا يوجد:
- Makima Shards
- Gojo Shards
- Naruto Shards
- أي شارد باسم شخصية

## التكرار
التكرار مسموح.

يعني ممكن يكون عند اللاعب أكثر من نسخة من نفس الشخصية.
كل نسخة تعتبر Card مستقل.

نفس النسخة الواحدة لا تدخل في تشكيلتين بنفس الوقت، لكن النسخ المكررة تقدر تدخل في تشكيلات مختلفة.

## الباسف
الباسف لا يتغير ولا ينقص بسبب التكرار.

كل نسخة تحتفظ بنفس Passive الشخصية.

- Tank: Guardian Sigil
- DPS: Slayer Sigil

- Summoner: Spirit Sigil

### Element Cores

- Fire: Flame Core
- Ice: Frost Core
- Water: Tide Core
- Wind: Gale Core
- Lightning: Storm Core
- Shadow: Shadow Core
- Light: Light Core
- Void: Void Core

### Rarity Materials
مواد حسب الندرة:
- Divine: Divine Fragment
- Voidborn: Voidborn Shard
- Secret: Secret Core

Duplicate Corrupted Makima gives:
- Soul Fragments
- Essence
- Gold
- Control Sigils
- Void Cores

وليس Makima Shards.
