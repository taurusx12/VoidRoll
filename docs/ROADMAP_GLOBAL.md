# Global Launch Roadmap

Phase 1: Closed Alpha
- 1 server / 100 players
- tune roll cooldown and rarity
- monitor DB logs

Phase 2: Economy Hardening
- trade logs
- fraud flags
- anti-spam/rate-limit per command
- rollback scripts for duplicated listings

Phase 3: Content
- limited banners
- season pass
- raids
- clans
- auctions
- fusion/awakening

Phase 4: Monetization
- sell cosmetics only first
- avoid pay-to-win early
- Stripe/PayPal checkout outside Discord
- receipts + entitlement webhooks

Phase 5: Global
- shard bot
- move images to CDN
- add observability: Sentry, Grafana, uptime monitor
