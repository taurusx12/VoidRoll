# Phase 15 Command Notes — World Bosses / Raids

## /world-boss
Shows active world boss.

## /raid
Shows active raids.

## /raid-attack
Attacks a selected raid boss.

Recommended options:
- boss: active boss id

## /raid-rank
Shows damage ranking.

## /raid-rewards
Shows reward tiers.

## Important
Raids use:
- Battle Engine
- Formation System
- Damage Ranking

Do not use:
- power-only checks
- item rolls
- relic pulls
- aura pulls
- stars
- fusion
