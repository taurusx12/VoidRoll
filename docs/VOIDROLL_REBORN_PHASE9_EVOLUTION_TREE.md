# VoidRoll Reborn — Phase 9 Character Evolution Tree

## الهدف
إضافة أساس شجرة تطوير الشخصية بدل النجوم والدمج ورولات الأيتمات.

## القرارات
لا يوجد:
- Fusion
- Stars
- Item Rolls
- Character Shards

التطوير كله من داخل الشخصية.

## فروع الشجرة

### Core Tree
يرفع:
- HP
- ATK
- DEF
- SPD

### Skill Tree
يطور:
- Skills
- Ultimate
- Passive strength
- Effect Chance

### Built-in Gear Tree
المعدات مدمجة داخل الشخصية:

Common → Rare → Epic → Legendary → Mythic → Divine → Voidborn → Secret

### Trait Tree
يفتح ويطور التريتات.

### Bond Tree
يفتح:
- Quotes
- Dialogues
- Team Attacks
- Team Ultimates
- Bond Buffs

### Transformation Tree
للتحولات:
- Bankai
- Gear 5
- Baryon Mode
- Hollow Form
- Void Form

### Variant Tree
للنسخ الخاصة:
- Corrupted
- Awakened
- Fallen
- Eclipse
- Voidborn
- True Form

## الأوامر المقترحة

### /character-tree
يعرض شجرة الشخصية.

### /upgrade
يطور فرع محدد من الشجرة.

Options:
- card
- branch: core / skill / gear / trait / bond / transformation / variant
