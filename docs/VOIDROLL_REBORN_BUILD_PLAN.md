## Rarity order
COMMON → RARE → EPIC → LEGENDARY → MYTHIC → DIVINE → VOIDBORN → SECRET

VOIDBORN is the rarity between DIVINE and SECRET.
SECRET stays the rarest tier.

## Secret reveal target
SECRET reveal flow:
1. Void Spark
2. Bottom Distortion
3. Quote Ascends
4. Secret Flash
5. Character Reveal

The reveal starts from the bottom and rises until the final character card appears.

## Variants
Corrupted is only one variant name. Future variant families can include:
- Corrupted
- Voidborn
- Awakened
- Transcendent
- Fallen
- Monarch
- Hollow
- Cursed
- Divine
- Abyssal

A variant is a full alternate character, not just a skin. It can have different rarity, passive, role, element, lore, quote, frame and aura.

## Character database rule
The imported MyAnimeList database may contain around 6000 characters, but the playable curated target is 1000 high-value characters.
All famous characters should remain searchable and should be prioritized for banners, variants, passives and special reveals.

## Removed systems
- Anime Weather System
- Anime Betrayal System
- Real-Time Server Events
