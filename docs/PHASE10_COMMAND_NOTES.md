# Phase 10 Command Notes — Trait System

## /traits
Shows traits.

Options:
- role: optional

Examples:
- /traits
- /traits role:CONTROL
- /traits role:DPS

## /trait
Shows one trait.

Example:
- /trait name:Void Walker

## /trait-unlock
Unlocks a trait from the Character Evolution Tree.

Recommended options:
- card: owned card

## /trait-upgrade
Upgrades current trait.

Recommended options:
- card: owned card

## Important
Trait unlock/upgrade uses:
- Gold
- Essence
- Trait Dust
- Trait Crystals
- Void Crystals for VOID/SECRET traits
- Void Sigils / Secret Cores for high tier traits

Do not add:
- Fusion
- Stars
- Character Shards
